# ratbot.py

The main version of rat, a Discord bot developed by ernieIzde8ski in Python. It is the only discord bot.

## Requirements

- Python 3.10

## Running

~~~bash
python -m pip install -r requirements.txt
pip install -U git+https://github.com/Rapptz/discord.py
python .
~~~
